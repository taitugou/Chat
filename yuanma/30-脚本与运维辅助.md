# 脚本与运维辅助

## Token 清理（排障用）
- 脚本：[clearToken.ps1](file:///f:/C/backend/clearToken.ps1#L1-L35)
- 作用：提示在浏览器控制台清除 `localStorage('token')` 并重新登录，用于排查“旧 token 对应用户不存在/状态异常”等问题。

目录：[backend/scripts/](file:///f:/C/backend/scripts/)
